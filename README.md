# Interactive Sales Dashboard

## What I Learned

In this project, I learned how to use Python libraries like Pandas, Seaborn, Matplotlib, and Plotly to analyze and visualize data. I also learned how different types of charts help present information in a clear and meaningful way. This project gave me practical experience in creating an interactive dashboard and working with real-world datasets.

## Project Overview

The purpose of this project is to analyze a sales dataset and present the results using different visualizations. The program reads the dataset, performs basic analysis, and creates multiple charts that help understand sales performance, product trends, and relationships between different variables.

## Setup Instructions

1. Install Python 3.x on your computer.
2. Install the required libraries using:

```
pip install -r requirements.txt
```

3. Place all project files in the same project folder.
4. Create a folder named `visualizations` to store the generated charts.
5. Run the program using:

```
python dashboard.py
```

## Code Structure

Project Files:

- `dashboard.py` – Main Python program for loading the dataset and creating visualizations.
- `customer_analysis.ipynb` – Jupyter Notebook containing the data analysis.
- `sales_data.csv` – Sales dataset used in the project.
- `requirements.txt` – List of required Python libraries.
- `README.md` – Project documentation.
- `visualizations/` – Folder containing the generated charts and screenshots.

## Technical Details

This project uses:

- Pandas to load and analyze the dataset.
- Seaborn to create statistical charts.
- Matplotlib to display and save visualizations.
- Plotly to build an interactive dashboard.

The dashboard includes:

- Box Plot
- Violin Plot
- Heatmap
- Histogram
- Scatter Plot
- Interactive Plotly Bar Chart
